Juho Leppäkangas

Data Analyst / Data Engineer (Cloud & Analytics)

Olen tietojenkäsittelyn tradenomi, joka on erikoistunut pilvipalveluihin ja datan käsittelyyn muunmuassa AWS-ympäristössä. Minulla on käytännön kokemusta datan keräämisestä, käsittelystä ja visualisoinnista osana oikeita projekteja, ja ymmärrän datan koko elinkaaren datan ingestoinnista analyysiin ja päätöksenteon tukemiseen. Harjoittelussani työskentelin AWS Security Lake -ratkaisun parissa, jossa hyödynsin Lake Formation -oikeuksienhallintaa, kirjoitin Athena SQL -kyselyitä ja rakensin QuickSight-dashboardeja. Visualisoin datasta poikkeamia ja haavoittuvuuksia, mikä auttoi tuomaan keskeiset havainnot selkeästi esille ja tukemaan päätöksentekoa. Teknisen osaamisen lisäksi minua motivoi erityisesti datan hyödyntäminen liiketoiminnan kehittämisessä. Pyrin ymmärtämään datan taustalla olevia ilmiöitä ja tuottamaan selkeitä oivalluksia, jotka tukevat päätöksentekoa.

Työharjoittelu

Harjoittelussa pääsin tekemään töitä AWS Security Lake -palvelun, Lake Formation -oikeuksien ja Athena SQL -kyselyiden parissa. Visualisoin AWS-tilien haavoittuvuuksia ja poikkeamia QuickSight-dashboardeilla, mahdollistaen nopeamman reagoinnin tietoturvariskeihin ja päätöksenteon tukemisen.

  • AWS Security Lake ja Lake Formation
  • Athena SQL -kyselyt
  • Quicksight dashboardien luonti ja visualisointi. (Quicksight ei tunnista sisäkkäisiä JSON-rakenteita ilman puhdistamista)
  • Monitorointi ja tietoturva CloudOps -näkökulmasta

Opinnäytetyö

Opinnäytetyössä tutkin AWS Datajärvien toimintaa. Rakensin automatisoidun dataputken (API → Lambda → datajärvi), jonka avulla kerätty ja analysoitu data visualisoitiin interaktiivisiin QuickSight-dashboardeihin.

  • AWS Datajärvet ja Lake Formation
  • Athena SQL
  • CSV-datasetin keruu ja analysointi
  • Sportradar API -datan automaattinen haku Lambda funktiolla, jota pyöräytetään kello 7:00 aamulla EventBridgen avulla
  • Interaktiiviset dashboardit
