Juho Leppäkangas

Junior Cloud Engineer & Data Engineer

AWS pilvipalveluihin erikoistunut tietojenkäsittelyn tradenomi. Harjoittelussa suoritin AWS CloudOps Associate -sertifikaatin, jonka myötä sain tärkeää tietoa AWS-ympäristön hallinnasta ja ylläpidosta, palveluiden monitoroinnista sekä AWS-palveluiden tietoturvasta.

Työharjoittelu

Harjoittelussa pääsin tekemään töitä AWS Security Lake -palvelun, Lake Formation -oikeuksien ja Athena SQL -kyselyiden parissa. Visualisoin haavoittuvuuksia ja tietoa Quicksightillä, jolloin dashboardit päivittyivät automaattisesti.

  • AWS Security Lake ja Lake Formation
  • Athena SQL -kyselyt
  • Quicksight dashboardien luonti ja visualisointi
  • Monitorointi ja tietoturva CloudOps -näkökulmasta

Opinnäytetyö

Opinnäytetyössä tutkin AWS Datajärvien toimintaa. Keräsin ja analysoin CSV-datasetin, hain päivittäin päivittyvää dataa Sportradar API:sta ja visualisoin tiedot Quicksight-dashboardeihin.

  • AWS Datajärvet ja Lake Formation
  • CSV-datasetin keruu ja analysointi
  • Sportradar API -datan automaattinen haku Lambda funktiolla, jota pyöräytetään kello 7:00 aamulla
  • Quicksight dashboardien visualisointi
